 Useful Plugins
- `@astrojs/sitemap`
- `@astrojs/rss`
- `@astrojs/image`

To install a plugin:
bash
npm install @astrojs/sitemap


Add to `astro.config.mjs`:
js
import { defineConfig } from 'astro/config';
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://your-site-url.com',
  integrations: [sitemap()]
});

 10. Building and Deploying

 Build Command
bash
npm run build

Output will be generated in the `dist/` folder.

 Deploy to Netlify
1. Create a new Netlify account.
2. Connect your GitHub repository.
3. Choose â€œAstroâ€ as your framework.
4. Set build command as `npm run build` and publish directory as `dist`.

 Deploy to GitHub Pages
1. Install GitHub Pages adapter:
   bash
   npm install @astrojs/github
   
2. Add to `astro.config.mjs`:
   js
   import github from '@astrojs/github';
   export default {
     adapter: github(),
     site: 'https://yourusername.github.io/'
   };
   
3. Push your code and deploy.
